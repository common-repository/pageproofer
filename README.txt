== Description ==

[PageProofer](https://pageproofer.com) is the easiest way to leave feedback and track issues directly on your website. During development every website has typos, broken links, missing images, and bugs. With PageProofer when you spot an issue on your website, you can instantly alert your team, right from your website.

## Features
- capture feedback right on your website
- attach screenshots, screen recordings and other attachments to each note
- full commenting on all notes
- works in all major browsers and on all popular devices
- integrates with a variety of popular project managements and bug tracking software including:
  - Asana
  - Basecamp
  - JIRA
  - Teamwork
  - Trello

== Installation ==

Visit our [getting setup guide](https://pageproofer.com/help/how-can-i-add-pageproofer-to-my-wordpress-site) for a video walk through.

1. Log in to the WordPress Admin panel.
2. Click on the Plugins tab and then click the Add New Plugin button.
4. In the search box, type PageProofer to find the plugin.
5. From the results, click the Install Now button next to PageProofer. The Plugin will install.
6. Once the installation is done, click the Activate button.
7. Go to the Settings menu and select PageProofer. 
8. Get your site api key from https://app.pageproofer.com for the specific site you are working on.
8. Add your website api key to the PageProofer Site Api Key	field and click the Save button.
10. Open your website in a new browser tab and click on the PageProofer icon to begin leaving feedback
